# Frete-Saving-Oportunidades
voltado à análise de oportunidade de frete dedicado em comparação com frete fracionado.  O aplicativo permite apoiar a operação logística na tomada de decisão por meio de:  Simulador manual de cenários.  Upload de base histórica em Excel.  Indicadores de oportunidades de frete dedicado.  Estimativa de savings.  Download da análise gerada.
